IPFS for NFTs

Quick Implementation Guide

What is IPFS?
InterPlanetary File System (IPFS) is a distributed storage network that provides decentralized, immutable file storage. For NFTs, IPFS stores metadata and media files off-chain while maintaining decentralization.

Why Use IPFS for NFTs?

  • Decentralization: No single point of failure
  • Immutability: Content-addressed storage prevents tampering
  • Cost-effective: Much cheaper than on-chain storage
  • Permanence: Files remain accessible as long as nodes host them
  • Bandwidth: Distributed access improves performance

Using IPFS Desktop

Step 1: Install IPFS Desktop

  1. Download from ipfs.io
  2. Install and run IPFS Desktop
  3. Wait for node to initialize

Step 2: Add Files

Via Desktop Interface
1. Open IPFS Desktop 2. Go to "Files" tab 3. Click "Import" and select your files 4. Copy the generated hash 5. Test access: ipfs://YOUR_HASH

Step 3: Create Metadata JSON

metadata.json
{ "name": "Luxembourg Property #123", "description": "Luxury apartment in Luxembourg City center", "image": "ipfs://QmYourImageHash", "attributes": [ { "trait_type": "Location", "value": "Luxembourg City" }, { "trait_type": "Size", "value": "150 sqm" }, { "trait_type": "Valuation", "value": "€850,000" }, { "trait_type": "Property Type", "value": "Residential" } ] }

Step 4: Upload Metadata

  1. Save the JSON as a file (e.g., metadata.json)
  2. Import this JSON file to IPFS Desktop
  3. Copy the metadata IPFS hash
  4. Use this hash in your NFT's tokenURI

Best Practices

File Organization:
  • Create folder structure: images/, metadata/, documents/
  • Use consistent naming conventions
  • Keep original files as backup
Metadata Standards:
  • Follow OpenSea metadata standards
  • Include all relevant attributes
  • Use proper image formats (PNG, JPG, WebP)
  • Optimize file sizes for faster loading
Important Considerations:
  • Pinning: Files need to be pinned to stay available
  • Gateway dependency: Consider multiple IPFS gateways
  • File permanence: Use pinning services for long-term storage
  • Backup strategy: Keep copies of IPFS hashes and files
